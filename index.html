speccytools.org
A collection for modern Software Development for ZX Spectrum
Es.pectrum (win)
Emulator that also supports Spectranet.
Fuse (win)
Fuse Emulator fork. Supports Spectranet. Has gdbserver (to enable, select "GDBServer...").
z88dk
The Development Kit for Z80 Computers. It allows you to compile C code, which then be debugged on source level. Comes with z88dk-gdb, a gdb-alike client to debug your program with Fuse or real hardware (emulator or hardware needs Spectranet).
Spectranet
Connect Your Spectrum to network.
